# Veges PK Guide

A read-only sidebar guide for the **Veges Group Bronzeman**. It lays out the
main PK build paths and tracks your quest progress against an in-game-aware
checklist. It does **not** automate any gameplay.

## Features

- **Build paths** — summaries for **1 Def Pure**, **40 Def**, **Zerker (45 Def)**,
  **Med (70/75 Def)**, plus a **Pivot Guide** explaining the build ladder.
- **Quest checklist** — the Pure and Zerker paths have an ordered, tickable
  quest list with a completion progress bar. Progress is saved per build via
  RuneLite's config, so it persists between sessions.
- **Auto-completion** — quests you've already finished are ticked automatically
  on login, and newly completed quests tick themselves as you finish them. It
  only ever *adds* completions, so your manual ticks are never cleared.
- **Skill requirements** — each quest shows the skill levels it needs, coloured
  green (met) or red (not met) against your real (un-boosted) levels, updating
  live as you train.

## Usage

Open the gold **V** icon in the RuneLite sidebar, pick a build from the
dropdown, and work down the checklist. Use **Reset this build's checklist** to
clear a build's saved progress.
